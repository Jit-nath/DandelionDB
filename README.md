# Learning Path

Recommended books for understanding this codebase:

1. [Effective Python, 2nd Edition](https://www.oreilly.com/library/view/effective-python-90/9780134854717/) by Brett Slatkin
2. [Fluent Python, 2nd Edition](https://www.oreilly.com/library/view/fluent-python-2nd/9781492056348/) by Luciano Ramalho
3. [Architecture Patterns with Python](https://www.oreilly.com/library/view/architecture-patterns-with/9781492052197/) by Harry Percival and Bob Gregory
4. [Database Internals](https://www.oreilly.com/library/view/database-internals/9781492040330/) by Alex Petrov
5. [Designing Data-Intensive Applications](https://www.oreilly.com/library/view/designing-data-intensive-applications/9781491903063/) by Martin Kleppmann
6. [The Rust Programming Language](https://nostarch.com/Rust) by Steve Klabnik and Carol Nichols
7. [Programming Rust, 2nd Edition](https://www.oreilly.com/library/view/programming-rust-2nd/9781492052586/) by Jim Blandy, Jason Orendorff, and Leonora F. S. Tindall
8. [High Performance Python, 2nd Edition](https://www.oreilly.com/library/view/high-performance-python/9781492055013/) by Micha Gorelick and Ian Ozsvald

Start with `Effective Python`, then read the `Fluent Python` chapters on Python's data model, classes, decorators, and metaprogramming.
